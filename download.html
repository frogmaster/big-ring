Download

Big Ring can currently be used on Windows (32 and 64 bits) and Linux. for Windows a zip-file can be downloaded which contains all you need for running Big Ring Indoor Video Cycling.

Download version 1.7.0 for Windows

The zip contains the EXE file and all DLLs needed for running Big Ring. Simply extract it to a directory and run big-ring.exe by double clicking it. For details on running Big Ring, please see the Instructions page.

For Linux, no binary builds are available. You can download the source for the latest release from GitHub.

Download source for release 1.7.0 (tar.gz)

Download source for release 1.7.0 (zip)

For an overview of all releases see the releases page on GitHub. See the project page for details on building from source.
